# Logistic Regression

This repository implements Logistic Regression from scratch, starting from its mathematical foundations and progressively translating each concept into code.

The goal is to develop a clear understanding of how logistic regression works internally, rather than relying on high-level libraries.

---

## Learning Objectives

### 1. Binary Classification
- Understand the intuition behind classification problems
- Implement the sigmoid function
- Build a basic logistic regression model
- Perform predictions using learned parameters

### 2. Cross-Entropy Loss
- Derive and implement the log loss (cross-entropy loss)
- Understand why it is used instead of MSE
- Visualize loss behavior during training
- Plot loss vs iterations

### 3. Evaluation Metrics
- Construct a confusion matrix
- Compute accuracy, precision, and recall
- Understand trade-offs between metrics
- Evaluate model performance on test data
